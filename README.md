# Räknabil.se

A Swedish car ownership cost calculator. Calculate the true total cost of owning a car — for both private individuals and companies.

## Features

- **Drivmedel** — El, Bensin, Diesel, Laddhybrid
- **Ägandetid** — 1–6+ years
- **Automatic residual value** based on fuel type and ownership period
- **Loan calculator** with 80% cap rule
- **Company car mode** — förmånsvärde, momsavdrag, skattemässigt avdrag
- **Visual cost breakdown** with bar charts

## Tech

- React + Vite

## Run locally

```bash
npm install
npm run dev
```
